# Godot 4 Chess ♟

**A fully functional 2D Chess engine built from scratch using Godot 4. This project features custom-coded move validation, checkmate logic, and original hand-drawn assets.**

## Features

  Complete Move Validation: Custom logic for all pieces, including Knights, Bishops, and sliding pieces (Rooks/Queens).

### Smart Game State:

  ***-Check & Checkmate Detection: The game recognizes when a King is under threat and when no legal moves remain.***  
  ***-Stalemate Logic: Handles draws when a player has no legal moves but isn't in check.***  
  ***-Pawn Promotion: Includes a dedicated UI menu to promote pawns when they reach the back rank.***  
  ***-Dynamic Camera: The view automatically rotates to face the current player's perspective.***  
  ***-Move Safety: Prevents players from making moves that would leave or put their own King in check.***
